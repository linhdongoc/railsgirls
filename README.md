`2.5.0`

`5.2.4.3`

## Configuration

```
$ git clone https://github.com/linhdongoc/railsgirls.git
$ cd railsgirls
$ bundle install
$ bundle exec rake db:setup
or
$ bundle exec rake db:reset
$ rails s
```

## Run the test suite with RSpec

Run all RSpec tests

```
$ bundle exec rspec
```
